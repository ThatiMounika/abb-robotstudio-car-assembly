# ABB RobotStudio Car Assembly Simulation

## Overview

This project was developed as part of a RobotStudio simulation assignment.

The objective was to create an automated production cell that assembles one car by placing the individual parts in their correct positions.

The simulation focuses on robot movement, part handling, production-cell design, and assembly logic. The physical joining method, such as welding or gluing, is outside the scope of the project.

## Project Requirements

The assignment required:

- Assembly of one complete car
- Creation of at least one Smart Component
- Use of RobotStudio modelling functionality
- Development of a production line or robot cell
- Placement of all car parts in their correct positions

## Project Approach

The car parts are introduced into the robot cell and handled by the robot.

The robot performs a sequence of pick-and-place operations to move each component to the correct assembly position.

The project was developed using a reverse-engineering approach by first positioning the complete car parts and then defining how the robot should assemble them.

## Features

- Automated car assembly sequence

## How to Open the Project

1. Install ABB RobotStudio.
2. Download or clone this repository.
3. Open `Robotstudio-car-assembly.rspag`.
4. Restore or load the project in RobotStudio.
5. Start the virtual controller if required.
6. Run the RAPID program or simulation.
